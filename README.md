A discord bot that shows website description.

## Basic usage

### Clone

```shell
git clone https://github.com/canaria3406/ermianaJS.git
```

### Settings

`.env`

```
DCTK="你的discord token"
BHUD="你的巴哈帳號"
BHPD="你的巴哈密碼"
```
